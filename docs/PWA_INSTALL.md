# PWA instalable

Publique `flutter build web --dart-define=API_BASE_URL=https://api.ejemplo.com/api` en HTTPS. En Android, abra Chrome, menú y **Agregar a pantalla principal**. En iPhone, abra Safari, **Compartir → Agregar a pantalla de inicio**. La instalación muestra el icono USCOM y abre como aplicación independiente cuando el navegador lo permite. La cámara solicita permiso del navegador.

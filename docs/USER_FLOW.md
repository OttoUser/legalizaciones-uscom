# Flujo de usuario

Inicie sesión, cree el proyecto con anticipo y moneda, agregue gastos y capture recibos. Se solicita confirmación del gasto y si desea otro. El resumen muestra totales COP/USD, conversión y saldo. Al finalizar se bloquea la edición y se habilitan PDF, Excel y el cliente de correo con autorización explícita de la persona usuaria.
